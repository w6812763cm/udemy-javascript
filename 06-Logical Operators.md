### 符號

* 比較符號

|比較符號|意思|
|:---:|---|
|  >  |大於|
| >=  |大於等於|
|  <  |小於|
| <=  |小於等於|
| ==  |等於|
| !=  |不等於|
| === |等於且型態(Type)符合   |
| !== |不等於或型態(Type)不符合|

* 邏輯判斷符號

|邏輯符號|意思|
|:---:|---|
| &&  |AND|
| \|\|  |OR |
| !   |NOT|

* 屬於false的值：
    * false
    * 0
    * null
    * undefined
    * ""(empty String)
    * NaN (Not a Number)
