### JavaScript 5 Primitive Datatypes
---
####五大型態
* Numbers
無論正數、負數、分數、小數都是屬於numbers，javascript不分
* String
* Boolean
* null
* undefined

##### 串聯文字
`"hello " + "world" > "hello world"`

##### 轉義字符(字串重複使用雙引號/單引號)
* 使用反斜線
```
"backflash is escape \"characters\" "
>"backflash is escape "characters" "
```
* 若要打反斜線要多加一個\
```
"Hello\\"
>"Hello\" 
```


##### 字串長度
```length
"hello".length
>5
```
##### 取得字串的獨立字元(使用[索引數字])
 * 索引數由0開始
```
"The Beatles" [0]
>"T"
```
